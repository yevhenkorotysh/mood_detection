Mood Detection Service from Audio Files

This project is aimed at building a Mood Detection Service from audio files. The goal is to create two services:

  1. Service to Store an Audio File: This service allows users to upload an audio file, which is then stored in a cloud storage system, like Azure Blob Store.
  2. Service to Retrieve Mood: The second service analyzes the uploaded audio file to determine the mood and returns one of three different moods: Happy, Sad, or Calm. The actual mood determination is not implemented in this project; instead, a random mood generator is used based on the time of the day.

Architecture

Language Choice

The service code can be implemented in the language of your choice, although Python is preferred due to its extensive libraries and ease of use for audio processing. In this example, we will use Python for illustration purposes.

Components

The architecture of the project involves several components:

  1. Audio File Storage: Cloud storage, such as Azure Blob Store, is used to store uploaded audio files. These files should be limited to a maximum duration of 30 seconds and can be in WAV or MP3 format.
  2. Audio Processing: To analyze the mood, the uploaded audio file is processed. In this example, a random mood is generated based on the time of the day. However, a real implementation would require audio analysis using libraries like Librosa or PyDub for feature extraction.
  3. Data Store: You may need a database to keep track of the uploaded audio files, their metadata, and the associated moods. A database design that suits your needs should be chosen.
  4. Developer Console Testing: Both services can be tested from the developer console without the need to create a public API. Testing instructions should be provided in the project documentation.

Cloud Services

Azure Blob Store: Azure Blob Storage is used to store the audio files securely and efficiently.

Database: The database can be hosted on a cloud service like Amazon RDS (Relational Database Service) or Azure SQL Database, depending on your cloud provider.
